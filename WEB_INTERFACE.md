# Web Interface Guide

## 🌐 Запуск веб-интерфейса

### Быстрый старт

```bash
py run.py
```

Эта команда:
- Запустит Flask сервер на порту 5000
- Автоматически откроет браузер с дашбордом
- Покажет адрес: http://localhost:5000

### Возможности интерфейса

#### 📊 Главная страница (Dashboard)

**Карточки статистики:**
- Общее количество собранных вакансий
- Средняя зарплата
- Наиболее востребованный навык
- Процент вакансий с указанной зарплатой

**Графики:**
- **Топ-20 ключевых слов** - столбчатая диаграмма наиболее частых слов в описаниях
- **Топ-15 навыков** - горизонтальная диаграмма востребованных навыков
- **Требования по опыту** - круговая диаграмма распределения по опыту

**Таблица вакансий:**
- Последние 10 собранных вакансий
- Название, компания, зарплата, опыт
- Прямые ссылки на вакансии на HH.ru

#### 🚀 Сбор данных

1. Нажмите кнопку **"Собрать вакансии"**
2. Появится статус-бар с прогрессом
3. Автоматическое обновление каждые 5 секунд
4. После завершения - автоматическое обновление дашборда

#### 📥 Экспорт данных

Кнопки экспорта позволяют скачать:
- **JSON** - все вакансии в структурированном формате
- **CSV** - таблица для Excel/Google Sheets

## 🎨 Особенности дизайна

- **Современный градиентный дизайн** с фиолетовыми оттенками
- **Адаптивная верстка** - работает на компьютерах, планшетах и телефонах
- **Плавные анимации** - карточки и кнопки с hover-эффектами
- **Интерактивные графики** - Chart.js с градиентами
- **Real-time обновления** - автоматическая проверка статуса

## 🔧 API Endpoints

Если хотите использовать API программно:

### GET /api/stats
Получить статистику и анализ

**Response:**
```json
{
  "success": true,
  "total_vacancies": 200,
  "report": {
    "top_keywords": [...],
    "top_skills": [...],
    "salary_stats": {...},
    "experience_stats": {...}
  }
}
```

### GET /api/vacancies
Получить список вакансий

**Parameters:**
- `page` - номер страницы (по умолчанию 1)
- `per_page` - количество на странице (по умолчанию 20)

**Response:**
```json
{
  "success": true,
  "vacancies": [...],
  "total": 200,
  "page": 1,
  "pages": 10
}
```

### POST /api/collect
Запустить сбор вакансий

**Response:**
```json
{
  "success": true,
  "message": "Collection started"
}
```

### GET /api/status
Проверить статус сбора

**Response:**
```json
{
  "running": false,
  "progress": 100,
  "message": "Completed! Collected 200 vacancies"
}
```

### GET /api/export/{format}
Скачать данные

**Formats:** `json`, `csv`

## 💡 Советы по использованию

1. **Первый запуск:** Сначала запустите сбор вакансий через командную строку или веб-интерфейс
2. **Обновление данных:** Нажмите "Обновить" для загрузки свежих данных без перезагрузки страницы
3. **Множественный запуск:** Можно открыть несколько вкладок с дашбордом
4. **Доступ из сети:** По умолчанию доступен на 0.0.0.0, можно открыть с других устройств в локальной сети

## 🛠️ Остановка сервера

Нажмите `Ctrl+C` в терминале где запущен сервер
